AdData Tools

Web-based tools for encrypting and decrypting AdData configurations, compatible with Unity AdDataEncryptionTool

🔒
Encryption Tool
Configure AdData settings and encrypt them using AES-256 encryption. Includes validation for MAX ad unit IDs and export functionality.
🔓
Decryption Tool
Decrypt encrypted AdData files or text. Load from files or paste encrypted data directly with automatic JSON validation.

Key Features

  • AES-256 encryption with PBKDF2 key derivation (10,000 iterations)
  • Compatible with Unity AdDataEncryptionTool encryption format
  • Real-time validation for MAX ad unit IDs (16 characters, lowercase letters and numbers)
  • Visual separation of ad types: Interstitial, Rewarded, and Banner
  • Dynamic array management for bidfloor configurations
  • Copy to clipboard and file export functionality
  • Fixed-size windows with scrollable content areas
  • Responsive design for desktop and mobile devices

These tools replicate the functionality of the Unity AdDataEncryptionTool and AdDataDecryptionTool.
